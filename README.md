# Tipe Data Number

- Ada dua jenis tipe data Number, yaitu :
  - Integer
  - Floating Point

1. Tipe Data Integer
   - int8
   - int16
   - int32
   - int64
   - uint8 (unsigned)
   - uint16
   - uint32
   - uint64
2. Tipe Data Floating
   - float32
   - float64
   - complex64
   - complex128

Alias

- byte alias untuk uint8
- rune alias untuk int32
- int alias untuk Minimal int32
- uint alias untuk Minimal uint32
